# AbusedLib
Lib mod used by most of my other mods for 1.14 that simplifies my life.

#Gradle Dependency

```
repositories {
    maven {
        name = "JitPack"
        url = "https://jitpack.io"
    }
}

dependencies {
    modCompile "com.github.abused:AbusedLib:master-SNAPSHOT"
}
```
